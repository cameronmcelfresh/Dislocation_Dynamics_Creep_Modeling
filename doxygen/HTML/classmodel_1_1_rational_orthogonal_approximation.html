Detailed Description

A quaternion-based implementation of the rational approximations of an orthogonal matrix representing a rotation.

Implementation from [1] Rational orthogonal approximations to orthogonal matrices Victor J. Milenkovic, Veljko Milenkovic Computational Geometry 7 (1997) 25-35
