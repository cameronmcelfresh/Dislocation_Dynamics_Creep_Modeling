Detailed Description

template<int _dim, int degree, template< typename T > class MappingType = IsoparametricMapping>
class model::DiscontinuousLagrangeElement< _dim, degree, MappingType >

Class template for a Lagrange finite element of degree p on a d-dimensional Simplex.

The degrees of freedom for a LagrangeElement are the functinal values computed at the nodes (LagrangeNode). In a d-dimensional Simplex, each node is identified by a set of (d+1) positive integers \(I_k\) which satisfy the Diophantine equation:

\[ \sum_{k=0}^{d+1}I_k=p \]

These integeres are found using the StarsAndBars algorithm. The baricentric (area) coordinate of node \(\{I_k\}\) are obtained as \(A_k=I_k/p\).

The shape function of node \(\{I_k\}\) are:

\[ N_{\{I_k\}}(A_0,\dots A_{d})=\prod_{k=0}^{d+1}\frac{1}{I_k!}\prod_{i=0}^{I_k-1}\left(pA_k-i\right) \]

This can also be written as:

\[ N_{\{I_k\}}(A_0,\dots A_{d})=\prod_{i=0}^{d+1}\left([c_{i0},\dots,c_{ip}]\cdot[A_i^0,\dots,A_i^p]\right) \]

The matrix \(c_{ij}\) is stored in the field sfCoeffs;
