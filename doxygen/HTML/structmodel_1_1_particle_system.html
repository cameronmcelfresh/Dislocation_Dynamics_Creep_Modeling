Detailed Description

template<typename _ParticleType>
struct model::ParticleSystem< _ParticleType >

Class template that stores _ParticleType objects organized in SpatialCell(s) and computes nearest-neighbor and far-field interactions among them.

ParticleSystem can be compiled in serial and parallel version, depending whether MODEL_MPI is defined or not.
